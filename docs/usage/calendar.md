# Calendar

The calendar shows events from the local calendar and your connected calendar integrations. You can add, edit, and delete events when the integration supports it.

## Views

Switch between **month**, **week**, **day**, and **agenda** views using the header. Optional week numbers (for shift events) can be toggled in [Settings]({{ '/usage/settings/' | relative_url }}) under client preferences.

## Events

- **Add:** Use the floating action button to create an event for the local calendar or integrations that support it.
- **Edit:** Open an event, expand the calendars section, and click the pencil icon (for supported integrations). Edit and click save.
- **Delete:** Open an event and use the dialog to remove it.

## Connecting calendars

Add calendar integration sources (iCal, Google Calendar, Shifts) under **Settings > Integrations** in the app. For setup, see [Integrations > Calendar]({{ '/integrations/calendar/' | relative_url }}).
