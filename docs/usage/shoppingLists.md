# Shopping Lists

Create and manage shopping lists and check off items as you shop. Lists can be synced with integrations such as Mealie or Tandoor.

## Lists 

- Native lists can be added using the floating action button
- Integration lists are synced once the associated integration is set up.

## Items

- Items can be added using the add button. Details (notes, quantity, etc.) vary by list type and integration.
- Items can be checked off, edited, or cleared.

## Integrations

Connect shopping services under **Settings > Integrations** for two-way sync. For setup, see [Integrations > Shopping]({{ '/integrations/shopping/' | relative_url }}).
