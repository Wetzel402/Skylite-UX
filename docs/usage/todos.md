# Todo Lists

Todo lists are organized by columns (for example per family member or workflow). Each item can have a due date and a priority.

## Columns

Lists are grouped into columns (Kanban-style). You can add, reorder, or remove columns.

- Columns are added automatically per user
- Additional columns can be added with the floating action button

## Items

Add, edit, check off, or delete items. You can set an optional due date, priority (e.g. Urgent, High, Medium, Low), and recurrence.

## Sort

The sort order (by date, priority, or A–Z) can be set in [Settings]({{ '/usage/settings/' | relative_url }}) under client preferences.
